
# Stellar Guardian Vault 🛡️🔐

Stellar Guardian Vault is an open-source **Account Abstraction** toolkit for Soroban. It solves the "Seed Phrase Problem" by allowing users to secure their funds with social recovery.

### Key Features:
- **Social Recovery:** Lost your phone? Your trusted Guardians can help you recover your account.
- **Spending Limits:** Set a daily maximum (e.g., 500 XLM) to protect against hacks.
- **Gasless Transactions:** Built to support future paymaster integrations.
